## TokenFactoryとの連携

LangBot は、ローカルにデプロイされたTokenFactoryと、TokenFactoryを使用して構築されたサードパーティのTokenFactoryサービスへの連携をサポートしています。

### 使用方法

1.  TokenFactoryからAPIキーを取得します
    ![获取 API key](/assets/langbot/get_api_key.png)

        ローカルにデプロイされたTokenFactoryの場合は、ご自身でAPIアドレスを設定してください（[コンテナネットワーク接続](https://docs.langbot.app/ja/workshop/network-details.html)を参照）。サードパーティのTokenFactoryサービスを使用する場合は、ページからアドレスをコピーできます。注意：アドレスの後に`/v1`を追加する必要があります。

2.  LangBotでモデルを追加し、TokenFactoryプロバイダーを選択し、対応するAPIキーとAPIアドレスを入力します
    ![添加 TokenFactory 模型](/assets/langbot/add_tokenfactory_model.png)

3.  パイプラインでモデルを選択します

    ![选择模型](/assets/langbot/select_model.png)

4.  会話デバッグで会話するか、パイプラインにバインドされたボットと会話することで使用できます

    ![对话](/assets/langbot/debug_chat.png)

    ![微信对话](/assets/langbot/wechat.png)

    ボットのデプロイと設定については、[ボットのデプロイ](https://docs.langbot.app/ja/deploy/platforms/readme.html)を参照してください。

### LangBotナレッジベースの使用

LangBot はTokenFactoryの埋め込みモデルの使用をサポートしており、これをナレッジベースのベクトルモデルとして利用できます。

1. LangBotで埋め込みモデルを追加し、TokenFactoryプロバイダーを選択します
   ![添加嵌入模型](/assets/langbot/add_embedding_model.png)

2. 新しいナレッジベースを作成する際に埋め込みモデルを選択します
   ![使用嵌入模型](/assets/langbot/use_embedding_model.png)

その他の使用方法については、LangBot公式ドキュメントを参照してください：[https://docs.langbot.app](https://docs.langbot.app/)
