---
title: 機能説明
---

1. 🎨 新しいUIインターフェース（一部のインターフェースはまだ更新待ちです）
2. 🌍 多言語対応（改善予定）
3. 🎨 [Midjourney-Proxy(Plus)](https://github.com/novicezk/midjourney-proxy)インターフェースのサポートを追加
4. 💰 オンラインチャージ機能に対応、システム設定で設定可能：
   - [x] 易支付
5.  ページネーションで1ページあたりの表示数を選択可能
6. 🔄 SQLite データベースストレージをサポート、すぐに使用可能、軽量で便利
7. 💵 モデルの回数制課金に対応、システム設定-運用設定で設定可能
8. ⚖️ チャネルの**加重ランダム**に対応
9. 📈 データダッシュボード（コンソール）
10. 🔒 トークンが呼び出し可能なモデルを設定可能
11. 🤖 Telegram認証ログインに対応：
    1. システム設定-ログイン登録設定-Telegramによるログインを許可
    2. [@Botfather](https://t.me/botfather)にコマンド/setdomainを入力
    3. あなたのボットを選択し、http(s)://あなたのウェブサイトアドレス/loginを入力
    4. Telegram Bot名は、bot usernameから@を削除した文字列です
12. 🎵 [Suno API](https://github.com/Suno-API/Suno-API)インターフェースのサポートを追加
13. 🔄 Rerankモデルに対応、現在CohereとJinaと互換性があり、Difyに接続可能
14. ⚡ **[OpenAI Realtime API](https://platform.openai.com/docs/guides/realtime/integration)** - OpenAIのRealtime APIに対応、Azureチャネルに対応
15. ルート/chat2linkを使用してチャットインターフェースに入ることに対応
16. 🧠 モデル名のサフィックスによるreasoning effortの設定に対応：
    1. OpenAI oシリーズモデル
       - サフィックス `-high` を追加して high reasoning effort に設定 (例: `o3-mini-high`)
       - サフィックス `-medium` を追加して medium reasoning effort に設定 (例: `o3-mini-medium`)
       - サフィックス `-low` を追加して low reasoning effort に設定 (例: `o3-mini-low`)
    2. Claude 思考モデル
       - サフィックス `-thinking` を追加して思考モードを有効化 (例: `claude-3-7-sonnet-20250219-thinking`)
17. 🔄 思考内容の変換、`チャネル-編集-チャネル追加設定`で`thinking_to_content`オプションを設定可能。デフォルトは`false`で、有効にすると思考内容`reasoning_content`が`<think>`タグに変換され、コンテンツに連結して返されます。
18. 🔄 モデルのレート制限、`システム設定-レート制限設定`でモデルのレート制限を設定可能。総リクエスト数制限と成功リクエスト数制限の設定に対応。
19. 💰 キャッシュ課金に対応、有効にすると、キャッシュヒット時に設定された比率で課金可能：
    1. `システム設定-運用設定`で「プロンプトキャッシュ比率」オプションを設定
    2. チャネルでプロンプトキャッシュ比率を設定（範囲は0-1）。例えば0.5に設定すると、キャッシュヒット時に50%で課金されることを意味します
    3. 対応チャネル：
    - [x] OpenAI
    - [x] Azure
    - [x] DeepSeek
    - [ ] Claude
